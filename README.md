## Description

[CordAPi]() helping APi for the konicord website.

## Setup

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# genereate Prisma Database 
$ npm run gen-typing

# start the watch mode for the sql prisma database
$ npm run prisma:generate

# test database query with John as a donation example
$ npm run seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
